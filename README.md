## Running Locally

1. Clone the repository
2. Navigate to the backend folder:
   ```bash
   cd part3/phonebook/backend
   npm install
   npm start
3. Navigate to the frontend folder:
  ```bash
    cd ../frontend
    npm install
    npm run dev
